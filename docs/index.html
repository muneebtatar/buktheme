Buk Documentation

An overview of the Buk theme along with usage instructions.

Introduction

Buk is a unique twitter bootstrap themes for developers and freelancers.

You have downloaded buk! These docs provide usage instructions for the theme. If you run into any other problems with the theme, or encounter any bugs, please contact Tutswoods directly through our website.

What's Included

Plugins

Buk comes with several javascript and css plugins. Some of these plugins act as dependencies for Buk.

  • Bookblock (Fullscreen) Bookblock is one of the major dependencies for buk. It is this javascript plugin that allows buk to have its page flip effect.
  • jScrollPane jScrollPane is another dependencies for buk. It is used as the default scrollbar for content in buk.
  • Interactive BG A jQuery plugin that will let you create an interactive moving background/object that reacts to viewer's cursor.
  • jquery-mousewheel A jQuery plugin that adds cross-browser mouse wheel support.
  • Chartjs Simple HTML5 Charts using the canvas tag.
  • Cycle2 Cycle2 is a versatile slideshow plugin for jQuery built around ease-of-use.
  • Animate.css A cross-browser library of CSS animations. As easy to use as an easy thing.

Digging deeper

You can also learn the mechanisms behind the css and javascript plugins that Buk uses so that you may use buk to the fullest, Though it is not necessary but if are serious about customizing buk then it is recommanded you go through each of these plugins - especially the Bookblock plugin, which is the backbone of Buk.

HTML structure

We will go through how buk markup is structured.

Main HTML structure

We have a main container(.containerbuk) for all our elements. Inside, There is a division for the sidebar with the class menu-panel, and the wrapper for the BookBlock plugin with the class bb-custom-wrapper. This wrapper contains a div(.bb-bookblock) to which the bookblock plugin has been applied. Inside the .bb-bookblock div we have divs for each page with the bb-item class.

<div id="containerbuk" class="containerbuk">	

 <div class="menu-panel">
 
  <h3>Buk</h3>
  
  <ul id="menu-toc" class="menu-toc">
   
   <li class="menu-toc-current"><a href="#hello">Hello</a></li>
   
   <li><a href="#whoami">Who am i</a></li>
   
   <li><a href="#whatido">What i do</a></li>
   
   <li><a href="#portfolio">Portfolio</a></li>
   
   <li><a href="#contact">Get in touch</a></li>
  
  </ul>
  
 </div>

 <div class="bb-custom-wrapper">
  
  <div id="bb-bookblock" class="bb-bookblock">
  
   <div class="bb-item" id="hello"></div>
  
   <div class="bb-item" id="whoami"></div>
  
   <div class="bb-item" id="whatido"></div>
  
   <div class="bb-item" id="portfolio"></div>
  
   <div class="bb-item" id="contact"></div>
  
  </div>
  
 </div>
 
</div>

Page HTML structure

Each page has its own div with the class bb-item. Inside of each page, we have content wrapper(.content) with a div(scroller) that we’ll need for the custom scroll functionality(jScrollPane).

<div class="bb-item" id="page1">
 <div class="content">
  <div class="scroller">
   <p>This is a page</p>
  </div>
 </div>
</div>

How to

How to use google maps with buk.

Using google maps with buk is fairly easy.

Remember, you can also add google maps the way you seem fit - this is not the only way, there are no restrictions.

  1. 1. Generate a map using the Google maps embed api from here, you will need a API key in order to do this, which you can get from the Google API console.
  2. 2. Once everying is done copy the generated code into the page where you want to use it, be sure to wrap it in a div with class of google-maps

How to use morphing devices

The idea behind morphing devices is to show a screenshot of a website or app by applying a “device class" while using the same elements and pseudo-elements for all the devices,

Using morphing devices in portfolio

Using morphing devices in the portfolio page is fairly simple. You need to change the value of the data-devicewrapper attribute to the device wrapper you want to use and also change the value of the data-largesrc attribute to the screenshot image URL.

data-devicewrapper value Function
imac Use imac as morphing device
macbook Use macbook as morphing device
ipad Use ipad as morphing device
iphone Use iphone as morphing device

How to use buttons

Using the buttons that come with buk is simple. There is a main button class known as .btn-theme and various other button color classes. In order to use a button you must first apply the main button class and then the color button class. Say for exmaple, you want to have a green button your code would be;

<div class="btn-theme btn-green">
 I'm a button
</div>

How to change landing background

To change the landing background of the 'hello' page you must change the value of the data-ibg-bg attribute - which is applied on the .bb-item div of the 'hello' page. The data-ibg-bg attribute must reflect the location of the image you want to use as your background. For example, if your image is on the root folder and is called "background.png", change the attribute to "/background.png".
